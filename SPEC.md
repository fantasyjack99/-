# 機房巡檢系統 - 專案規格書

## 1. 系統架構

```
┌──────────────────┐     ┌───────────────────┐     ┌─────────────────┐
│   HTML/CSS/JS    │ ──▶ │  Google App Script │ ──▶ │ Google Sheets  │
│   (巡檢網頁前端)  │     │   (後端 API)        │     │  (資料庫)       │
└──────────────────┘     └───────────────────┘     └─────────────────┘
                                │
                                ▼
                         ┌───────────────────┐
                         │   Slack Webhook    │
                         │   (每週通知)        │
                         └───────────────────┘
```

## 2. 功能清單

### 2.1 認證
- [x] Google 帳號登入（Session 管理）
- [x] 權限檢查（只有授權帳號可使用）

### 2.2 巡檢功能
- [x] 顯示當日巡檢項目（依星期自動調整）
- [x] 勾選正常/異常
- [x] 異常時可填寫備註說明
- [x] 顯示進度（已勾選/總項目）
- [x] 一鍵提交儲存

### 2.3 通知
- [x] 每週五巡檢完成後觸發 Slack Webhook
- [x] 異常項目特別提醒

### 2.4 介面
- [x] RWD 響應式設計（手機/電腦皆可用）
- [x] 現代化視覺設計
- [x] 進度條顯示
